# OpenClaw Demo

This repository contains the configuration for an OpenClaw instance.

## About OpenClaw

OpenClaw is an open-source autonomous AI personal assistant that runs locally and integrates with messaging platforms.

## Setup

1. Install OpenClaw (already done via Homebrew)
2. Run onboarding: `openclaw onboard --install-daemon`
3. Configure your Claude API key
4. Start the gateway

## Configuration

OpenClaw workspace is located at `~/.openclaw/workspace`
